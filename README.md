# SkyIndia Airport Experience

Ultra-premium **frontend-only** airport and aviation platform built with React 19, Vite, Tailwind CSS v4, TypeScript, Framer Motion, GSAP, React Three Fiber, Radix UI (Shadcn-style), React Router, and React Query.

## Quick Start

```bash
cd "SkyIndia Airport Experience"
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
npm run preview
```

## Features

- **Landing** — Cinematic hero, flight search, destinations, live flights, services, lounges, analytics preview, testimonials, news, FAQ
- **Flights** — One-way / round-trip / multi-city search with flight cards
- **Flight Tracker** — SVG world map, animated routes, radar simulation, sidebar timeline
- **Destinations** — 8 Indian cities with weather, attractions, hotels
- **Airport Services** — Service cards, booking modal, comparison table
- **Terminal Guide** — Interactive map, zoom, search, shortest path
- **Lounges** — Cards, virtual tour modal, membership tabs
- **Dashboard** — Analytics widgets, Recharts (area, line, bar, pie), heat map
- **AI Chat** — Mock travel assistant with suggested prompts
- **Theme** — Dark/light mode, glassmorphism, custom cursor, particles, Lenis smooth scroll, GSAP ScrollTrigger

## Stack

React 19 · Vite 8 · Tailwind v4 · TypeScript · Framer Motion · GSAP · R3F · Lucide · React Query · React Router

## Note

All data and bookings are **mock/demo** only. No backend required.
